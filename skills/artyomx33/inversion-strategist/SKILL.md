# 反转策略师

## 核心原则

"反转，始终反转。" - 卡尔·雅可比

不问：我如何实现X？
而是问：什么会保证我**不**实现X？
然后：系统性地避免这些事情。

## 工作流程

1. **陈述目标**
2. **反转它：** "什么会保证失败？"
3. **列出失败路径**（7-10个，明显和微妙的）
4. **分类：** 致命 / 损害 / 烦恼
5. **翻转为避免策略**
6. **构建反检查清单**

## 输出格式

```
目标：[你想要什么]
反转："我如何在[目标]上肯定失败？"

失败路径：

🔴 致命：
• [路径] → 避免：[策略]

🟡 损害：
• [路径] → 避免：[策略]

🟢 烦恼：
• [路径] → 避免：[策略]

反检查清单：
□ 从不 [行为]
□ 从不 [行为]

正向检查清单：
□ 总是 [失败的相反]
□ 总是 [失败的相反]
```

## 查理·芒格智慧

"值得注意的是，我们这些长期优势的人是如何通过持续不愚蠢来获得的，而不是试图非常聪明。"

"告诉我我将死在哪里，我就永远不会去那里。"

## 组合使用

与以下技能协同增效：
- **first-principles-decomposer** - 从基本原理重建后，反转以找出新方法的风险
- **pre-mortem-analyst** - 反转找到路径；预死分析时间线压力测试
- **six-thinking-hats** - 黑帽是轻量级反转；组合进行全面分析
