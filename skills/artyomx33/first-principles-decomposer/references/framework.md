# 第一性原理框架 - 详细方法论

## 埃隆·马斯克方法（物理思维）

"把事情简化到最基本的真理，然后从那里推理上去。"

### 电池示例
- **传统思维**：电池很贵（600美元/千瓦时），一直如此
- **第一性原理**：电池由什么组成？碳、镍、铝、聚合物
- **原子问题**：这些材料在伦敦金属交易所的价格是多少？
- **发现**：原材料 = 80美元/千瓦时
- **重建**：工程改进制造 → 特斯拉的成本突破

## 分解阶梯

### 第一层：表面问题
"我们需要更好的家长参与度"

### 第二层：功能需求
"家长需要感到与孩子的一天有连接"

### 第三层：情感核心
"家长对不在场感到内疚"

### 第四层：根本真理
"人类需要为他们所爱的人提供幸福的证明"

### 第五层：原子解决方案
照片 + 时间戳 + 孩子的脸 = 幸福的证明
其他一切都是装饰。

## 常见假设陷阱

### "行业标准"陷阱
"每个人都这样做" → 为什么每个人都这样做？
通常：遗留问题，而非逻辑

### "客户说"陷阱
"客户要求X" → 他们试图解决什么问题？
通常：他们在描述解决方案，而非需求

### "技术存在"陷阱
"我们应该使用[技术]，因为它存在" → 这个根本需求是什么？
通常：寻找问题的解决方案

### "竞争对手在做"陷阱
"竞争对手有这个功能" → 它真的解决实际问题吗？
通常：功能膨胀，而非价值

## 穿透性提问

1. "如果今天从零开始，我们会这样构建吗？"
2. "一个完全的外人会觉得什么奇怪？"
3. "如果这贵10倍，我们还会做吗？"
4. "如果这是非法的，我们会做什么？"
5. "一个孩子能理解的版本是什么？"

## 与其他技能的组合

- 使用**反转**来验证："这个根本解决方案如何失败？"
- 使用**二阶效应**来预测："实施这个之后会发生什么？"
- 使用**预死分析**之前构建："想象这失败了。为什么？"
