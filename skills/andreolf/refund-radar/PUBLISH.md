# 退款雷达

## 元数据

- **名称**：refund-radar
- **版本**：1.0.0
- **作者**：andreolf
- **仓库**：https://github.com/andreolf/refund-radar
- **许可证**：MIT

## 描述

本地优先的银行对账单审计器，可检测定期收费、标记可疑交易，并起草可发送的退款请求模板。生成带有隐私开关的交互式 HTML 报告。

## 标签

- 财务
- 银行
- 对账单
- 退款
- 订阅
- 审计
- 隐私
- 本地优先

## 功能

- 使用自动检测功能解析 CSV 和文本银行对账单
- 检测带有周期估计的定期订阅
- 标记重复项、金额异常、新商户、费用
- 生成可发送的退款模板（电子邮件、聊天、争议）
- 带有深色/浅色模式的交互式 HTML 报告
- 隐私开关以模糊商户名称
- 持久状态用于学习偏好
- 零外部依赖

## 工作流程摘要

1.用户提供银行对账单（CSV 或粘贴的文本）
2.工具解析和标准化交易
3.自动检测定期收费
4.标记可疑/意外收费
5.分批询问确认问题
6.生成交互式 HTML 报告
7.起草退款请求模板
8.从用户决策中学习以供未来运行

## 要求

- Python 3.9+
- 无需外部包

## 隐私

所有处理都在本地进行。无网络调用。无外部 API。

## 屏幕截图

报告显示：
- 带有总计的摘要卡片
- 带有下次预期日期的定期收费表
- 带有严重程度徽章的标记交易
- 复制到剪贴板的退款模板
- 深色模式和隐私模糊切换
