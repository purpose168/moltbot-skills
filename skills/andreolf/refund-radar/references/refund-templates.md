# 退款模板参考

模板仅针对高严重程度和中严重程度的标记生成。
所有模板根据规范避免撇号。

## 模板类型

### 电子邮件
带有主题行的完整正式电子邮件。适用于：
- 与商户的首次联系
- 文档记录
- 较大金额

### 聊天
用于实时支持的简短消息。适用于：
- 快速解决
- 实时支持
- 较小金额

### 争议
正式的银行争议表单文本。适用于：
- 未经授权的收费
- 商户解决失败
- 欺诈案件

## 语气变体

### 简洁（默认）
- 直接且专业
- 清晰地陈述事实
- 礼貌地请求行动

### 坚定
- 断言性语言
- 参考消费者权利
- 设定最后期限
- 提及升级选项

### 友好
- 温暖且礼貌
- 假设善意
- 请求帮助
- 预先致谢

## 模板占位符

模板包含这些占位符供用户填写：

| 占位符 | 描述 |
|--------|------|
| `[YOUR NAME]` | 客户姓名 |
| `[CARD ENDING IN XXXX]` | 卡号后 4 位 |
| `[YOUR PHONE NUMBER]` | 联系电话 |
| `[IF AVAILABLE]` | 参考编号（如已知） |

## 按标记类型的原因文本

### 重复
> 这看起来是一笔重复收费。我因同一笔交易被扣了两次款。

### 金额异常
> 这笔收费比我通常在该商户的收费高出很多。我认为可能存在计费错误。

### 新商户
> 我不认可此商户，也未授权此笔交易。

### 费用类
> 此费用未被披露或同意。我请求退还此笔收费。

### 货币异常
> 此笔交易以意外货币处理。我可能遭受了未经授权的货币转换。

### 缺失退款
> 我之前对此笔收费有争议，但尚未收到退款。我正在跟进此事。

## 争议类别

| 标记类型 | 争议类别 |
|---------|---------|
| 重复 | 重复交易 |
| 金额异常 | 金额错误收费 |
| 新商户 | 未经授权的交易 / 欺诈 |
| 费用类 | 未经授权的费用 |
| 货币异常 | 货币/转换错误 |
| 缺失退款 | 未收到退款 |

## 电子邮件示例（简洁）

```
主题：退款请求 - 2026年1月12日的 29.99 美元收费

您好，

我请求退还以下交易：

商户：Amazon
金额：29.99 美元
日期：2026年1月12日

原因：这看起来是一笔重复收费。我因同一笔交易被扣了两次款。

请在您方便时尽快处理此退款。

谢谢。
```

## 聊天示例（坚定）

```
我需要立即争议一笔收费。2026年1月12日从 Amazon 收取的 29.99 美元。这是一笔重复收费。我需要今天退款，否则我将提交正式争议。
```

## 争议示例（友好）

```
您好，

我想就我账户上的一笔收费提交争议。

详情如下：
- 商户：Amazon
- 金额：29.99 美元
- 日期：2026年1月12日
- 我的卡号末四位：[XXXX]

争议原因：重复交易

情况说明：这看起来是一笔重复收费。我因同一笔交易被扣了两次款。

如果您能调查此事并帮助我获得退款，我将不胜感激。如需任何其他信息，请告诉我。

感谢您的帮助！

[YOUR NAME]
[YOUR PHONE NUMBER]
```
