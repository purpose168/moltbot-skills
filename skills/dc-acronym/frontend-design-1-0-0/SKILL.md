此技能指导创建独特的、生产级的前端界面，避免通用的"AI 垃圾"美学风格。实现真正可用的代码，特别注重美学细节和创意选择。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能会包含有关目的、受众或技术约束的上下文信息。

## 设计思考

在编码之前，了解上下文并确定一个大胆的美学方向：
- **目的**：此界面解决什么问题？谁使用它？
- **基调**：选择一个极端风格：极简主义、 maximalist 混乱、复古未来主义、有机/自然、奢华/精致、 playful/玩具般、编辑/杂志风格、 brutalist/原始、艺术装饰/几何、柔和/ pastel、工业/实用主义等。有很多风格可供选择。使用这些作为灵感，但设计一个真正符合美学方向的作品。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让这变得令人难忘？人们会记住的一件事是什么？

**关键**：选择明确的概念方向并精确执行。大胆的 maximalism 和精致的 minimalism 都有效 - 关键是 intentionality，而不是强度。

然后实现工作代码（HTML/CSS/JS、React、Vue 等），它应该是：
- 生产级且功能齐全
- 视觉上引人注目且令人难忘
- 具有清晰美学观点的凝聚力
- 每个细节都精心打磨

## 前端美学指南

重点关注：
- **排版**：选择美观、独特、有趣的字体。避免通用字体如 Arial 和 Inter；而是选择能提升前端美学的独特字体；意想不到的、有个性的字体选择。将独特的展示字体与精致的正文字体配对。
- **颜色和主题**：坚持一致的美学风格。使用 CSS 变量保持一致性。具有鲜明强调色的主色调优于胆怯的、均匀分布的调色板。
- **动效**：使用动画效果和微交互。优先为 HTML 使用纯 CSS 解决方案。在可用时为 React 使用 Motion 库。关注高影响力时刻：一个精心编排的页面加载，带有交错显示（animation-delay），比分散的微交互更能创造愉悦感。使用滚动触发和令人惊喜的悬停状态。
- **空间构图**：意想不到的布局。不对称。重叠。对角线流动。打破网格的元素。充足的负空间或受控的密度。
- **背景和视觉细节**：创造氛围和深度，而不是默认为纯色。添加与整体美学匹配的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒覆盖。

永远不要使用通用的 AI 生成美学，如过度使用的字体系列（Inter、Roboto、Arial、系统字体）、陈词滥调的配色方案（特别是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定特征的千篇一律的设计。

创造性地解释并做出意想不到的选择，让人感觉是真正为上下文设计的。没有设计应该是相同的。在明暗主题、不同字体、不同美学之间变化。永远不要在不同代际之间收敛于常见选择（例如 Space Grotesk）。

**重要**：将实现复杂性与美学愿景相匹配。Maximalist 设计需要精心制作的代码，具有广泛的动画和效果。极简主义或精致的设计需要克制、精确，并仔细关注间距、排版和微妙的细节。优雅来自于很好地执行愿景。

记住：Claude 有能力创造非凡的创意作品。不要退缩，展示当跳出框框思考并完全致力于独特愿景时真正可以创造的东西。
